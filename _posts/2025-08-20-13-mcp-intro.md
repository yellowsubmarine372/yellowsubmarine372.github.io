### Hands-on MCP Servers

Normally a poject will implement either an MCP client or an MCP server(built server to use 3'rd persons to use it in agents, or client --like us-- to use that service in building agents) but in this hands-on, we're making all of it

- Server

```python
from mcp.server.fastmcp import FastMCP

mcp = FastMCP("DocumentMCP", log_level="ERROR")
```
Building an MCP server becomes much simpler when you use the official Python SDK. Instead of writing complex JSON schemas by hand, you can define tools with decorators and let the SDK handle the heavy lifting.
And writes decorator
```python
@mcp.tool(
    name="read_doc_contents",
    description="Read the contents of document and return it as a string"
)
```

```python
@mcp.resource(
    "docs://documents",
    mime_type="application/json" # resource can return any type of data, give hint what we are returning
)
def list_docs() -> list[str]:
    return list(docs.keys())
```
